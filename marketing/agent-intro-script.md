# OnHyper: Agent Introduction Video Script

**Target Audience**: AI agents, developers, and agent harness users  
**Duration**: 45-50 seconds  
**Tone**: Conversational, energetic, developer-friendly

---

## SCRIPT

### [0:00-0:05] HOOK
**VISUAL**: Quick montage - code snippets, agent interfaces, "Hello World" apps spawning rapidly

**VOICEOVER**:  
*"What if you could go from agent output to live web app... in seconds?"*

---

### [0:05-0:15] MAIN VALUE PROP
**VISUAL**: Split screen - agent harness interface on left, live SPA appearing on right. API keys lock icon appears and glows green.

**VOICEOVER**:  
*"With your favorite agent harness, generate a single-page app that talks to your Scout OS backend or any LLM API. Then publish and serve on Hyper—fast. And your API keys? They stay safe. Always."*

---

### [0:15-0:40] 4-STEP WALKTHROUGH
**VISUAL**: Clean, animated step cards appearing sequentially with checkmarks

**VOICEOVER**:  
*"Here's how it works:*

*One—create a Hyper account.  
*Two—add your API keys securely.  
*Three—build your SPA using the builder skill.  
*Four—publish to Hyper. That's it!"*

*[Beat]*  
*"Your app. Live. Ready for users."*

---

### [0:40-0:50] CALL TO ACTION
**VISUAL**: OnHyper logo + website URL. Developer typing at terminal, app deploys, celebration animation.

**VOICEOVER**:  
*"Stop hacking deployment. Start shipping. Get started at onhyper.io—your agents will thank you."*

---

## PRODUCTION NOTES

### Suggested Visual Elements:
- **Hook**: Fast cuts, energetic music build
- **Value Prop**: Split-screen effect, security lock animation
- **Steps**: Clean typography, numbered cards with satisfying "ding" sounds
- **CTA**: Logo lockup, URL prominent, upward/deploy animation

### Pacing Notes:
- Keep sentences punchy—developer audience appreciates efficiency
- Step cards should appear quickly but be readable
- Final CTA should feel energetic but not rushed

### Word Count: ~175 words (fits comfortably in 45-50 seconds leaving room for pacing)
